## Project: File Organizer Script

A Python script that automatically organizes files in a folder into subfolders based on file type (images, documents, audio, videos).

**Concepts used:**
- File system navigation
- Loops and conditionals
- Python standard library (`os`, `shutil`)

### Bulk File Renamer

A Python script that renames all files in a folder using a consistent naming pattern.

**Concepts used:**
- Looping through files
- String formatting
- File renaming with Python
