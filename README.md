# Python Automation Basics

This project is a learning-focused repository created to understand the fundamentals of Python programming through small, hands-on scripts.

## What this project covers

- Basic Python syntax and control flow
- Working with files and folders
- Writing small automation-style scripts
- Understanding and using Git for version control

## Current status

This project is currently in progress and is being built as part of a structured learning process. The scripts in this repository are intentionally simple and focus on clarity and understanding rather than advanced patterns or optimisations.
